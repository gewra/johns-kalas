# React Invitation SPA

This project is a single-page React invitation built with Vite. It renders one presentational event invitation screen with editable placeholder copy.

## Scripts

- `npm run dev` starts the Vite development server.
- `npm run build` creates a production build in `dist`.
- `npm run preview` serves the production build locally.

## Customizing the Invitation

Update the event copy in `src/App.jsx` to change the title, date, location, and other invitation details.
